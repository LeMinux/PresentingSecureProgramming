## Linux

Welcome to files on Linux!
Here you can learn about not just about Linux files, but also the file systems of Linux.
If you are more concerned about you as a programmer using files then the LinuxFiles.md is more practical for you.
However, if you want to learn more about how the Linux abstraction works, the extended file system, or just want to learn the LinuxFileSystems.md is of interest.
You can also think of the LinuxFileSystems.md as a sort of prequel that introduces how files come to be for you to use in the first place.
Basically I had to split these into two because the file systems of Linux was taking away from the main point of handling file securely, but is still relevant for people who like forensics.
I have even done a little bit of forensics from NSA code breaker challenges, so knowledge of how a file system works on a deeper level is still useful.
Although, I do recognize that the entire inner workings may not interest people, and I don't want to subject them to what is a section of its own just to get to file security.
